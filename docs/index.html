    Suomen kielessä verbeillä on monta vartaloa

    Minä keksin vartalojen nimet

    Passiivivartalo &

    Sanakirjamuodosta:

    1. -(e/i/O/U)A → -(e/i/O/U)tt-
      -AA → -ett-
    2. -dA → -t-
    3. -(ll/nn/rr/st) → -(l/n/r/s)t-
    4. 4-6. tyypit: -VtA → -Vtt-

    Vahvavartalo &

    Sanakirjamuodosta:

    1. -VA → -V-
    2. -dA → -∅-
    3. -(ll/nn/rr/st)A → -(l/n/r/s)e-
    4. -(A/O/U)tA → -(A/O/U)A-
    5. -itA → -itse-
    6. -etA → -ene-

    Heikkovartalo &

    Heikko vain 1. tyypissä! Muut tyypit jäävät vahvoihin.

    Sama (paitsi KPT-vaihtelua) kuin vahvavartalo.

    Infinitiivivartalo &

    Sanakirjamuodosta:

    1. -VA → -V-
    2. -dA → -∅-
    3. -(ll/nn/rr/st)A → -(l/n/r/s)-
    4. 4-6. tyypit: -tA → -t-

    Partisiipit &

    Aktiivin partisiipin preesens -vA &

    → vahvavartalo + -vA

    Esimerkit
    1. oppia → oppiva
    2. juoda → juova
    3. purra → pureva
    4. kadota → katoava
    5. tarvita → tarvitseva
    6. vanheta → vanheneva
    vA-partisiipin käytön esimerkit
    • itkevä lapsi = lapsi, joka itkee
    • laulava mies = mies, joka laulaa

    Aktiivin partisiipin perfekti -nUt &

    1. -A → -nUt
    2. -dA → -nUt
    3. -(ll/nn/rr/st)A → -(ll/nn/rr/ss)Ut
    4. 4-6. tyypit: -tA → -nnUt
    Esimerkit
    1. oppia → oppinut
    2. juoda → juonut
    3. purra → purrut
    4. kadota → kadonnut
    5. tarvita → tarvinnut
    6. vanheta → vanhennut
    nUt-partisiipin käytön esimerkit
    • koomassa ollut potilas = potilas, joka oli koomassa
    • piirtänyt lapsi = lapsi, joka piirsi

    nUt-partisiipin taivutus &

    Ei ole KPT-vaihtelua

    Yksikkö:

    • Partitiivi: -Ut → -UttA
    • Illatiivi: -Ut → -eeseen
    • Muut sijamuodot: -Ut → -ee- + sijapääte

    Monikko:

    • Partitiivi: -Ut → -eitA
    • Genetiivi: -Ut → -ei(d/tt)en
    • Illatiivi: -Ut → -ei(siin/hin)
    • Muut sijamuodot: -Ut → -ei- + sijapääte

    Passiivin partisiipin preesens -tAvA &

    → passiivivartalo + -AvA

    Esimerkit
    1. oppia → opittava
      alkaa → alettava
    2. voida → voitava
    3. jutella → juteltava
    4. pelätä → pelättävä
    5. tarvita → tarvittava
    6. pidetä → pidettävä

    nUt-partisiipin käytön esimerkit vain venäjän kielellä

    Passiivin partisiipin perfekti -tU &

    → passiivivartalo + -U

    Esimerkit
    1. oppia → opittu
      alkaa → alettu
    2. voida → voitu
    3. jutella → juteltu
    4. pelätä → pelätty
    5. tarvita → tarvittu
    6. pidetä → pidetty
    tU-partisiipin käytön esimerkit
    • valittu johtaja = johtaja, joka valittiin
    • soitettu biisi = biisi, joka soitettiin

    Agenttipartisiippi -mA &

    → vahvavartalo + -mA

    Esimerkit
    1. oppia → oppima
    2. juoda → juoma
    3. purra → purema
    4. kadota → katoama
    5. tarvita → tarvitsema
    6. vanheta → vanhenema
    mA-partisiipin käytön esimerkit
    • aloittamani kurssi = kussi, jonka aloitin
    • kirjoittamani kirja = kirja, jonka kirjoitin
