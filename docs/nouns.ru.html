Типы склонения

Институт исконных языков Финляндии (Kotus) представил разделение всех имён на 51 тип склонения. Каждый тип представляет из себя правила образования следующих особых форм: номинатив ед.ч., генетив мн.ч., партитив, иллатив, W1, W2 и S1, S2
