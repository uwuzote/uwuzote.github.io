    Основы &

    В финском языке у глаголов много основ

    Названия основ мои собственные

    Пассивная основа &

    Из словарной формы:

    1. -(e/i/O/U)A → -(e/i/O/U)tt-
      -AA → -ett-
    2. -dA → -t-
    3. -(ll/nn/rr/st) → -(l/n/r/s)t-
    4. 4-6. типы: -VtA → -Vtt-

    Сильная основа &

    Из словарной формы:

    1. -VA → -V-
    2. -dA → -∅-
    3. -(ll/nn/rr/st)A → -(l/n/r/s)e-
    4. -(A/O/U)tA → -(A/O/U)A-
    5. -itA → -itse-
    6. -etA → -ene-

    Слабая основа &

    Только для первого типа действительно в слабой ступени! Остальные типы остаются в сильной ступени.

    Совпадает (кроме KPT-чередования) с сильной основой.

    Инфинитивная основа &

    Из словарной формы:

    1. -VA → -V-
    2. -dA → -∅-
    3. -(ll/nn/rr/st)A → -(l/n/r/s)-
    4. 4-6. типы: -tA → -t-

    Причастия &

    Активное причастие настоящего времени vA &

    → сильная основа + -vA

    Примеры
    1. oppia → oppiva
    2. juoda → juova
    3. purra → pureva
    4. kadota → katoava
    5. tarvita → tarvitseva
    6. vanheta → vanheneva
    Примеры использования vA-причастия
    • itkevä lapsi (плачущий ребёнок) = lapsi, joka itkee (ребёнок, который плачет)
    • laulava mies (поющий мужчина) = mies, joka laulaa (мужчина, который поёт)

    Активное причастие прошлого времени nUt &

    1. -A → -nUt
    2. -dA → -nUt
    3. -(ll/nn/rr/st)A → -(ll/nn/rr/ss)Ut
    4. 4-6. типы: -tA → -nnUt
    Примеры
    1. oppia → oppinut
    2. juoda → juonut
    3. purra → purrut
    4. kadota → kadonnut
    5. tarvita → tarvinnut
    6. vanheta → vanhennut
    Примеры использования nUt-причастия
    • koomassa ollut potilas (находившийся в коме пациент) = potilas, joka oli koomassa (пациент, который был в коме)
    • piirtänyt lapsi (рисовавший ребёнок) = lapsi, joka piirsi (ребёнок, который рисовал)

    Склонение nUt-причастия &

    Нет KPT-чередования

    Единственное число:

    • Партитив: -Ut → -UttA
    • Иллатив: -Ut → -eeseen
    • Остальные падежи: -Ut → -ee- + окончание

    Множественное число:

    • Партитив: -Ut → -eitA
    • Генетив: -Ut → -ei(d/tt)en
    • Иллатив: -Ut → -ei(siin/hin)
    • Остальные падежи: -Ut → -ei- + окончание

    Примеры использования tAvA-причастия

    Все примеры переведены с uusikielemme.fi

    1. Что-то происходит с предметом:
      [Käännettävä kirja] on pitkä. = [Книга, которую переводят,] длинная.
      [Koulussa käytettävä kirja] on vanha. = [Используемая в школе книга] старая.
      Olen asunut [remontoitavassa talossa] vuoden. = Я прожил(а) год в реновируемом доме.
    2. Что-то можно сделать с преметом:
      [Kannettava tietokone] on painava. = [Переносной компьютер] (=ноутбук) тяжёлый.
      Se on [ymmärrettävä selitys]. = Это [понятное обьяснение].
      Se ei ole [uskottava valhe]. = Это не [правдоподобная ложь].
      Osaatko tunnistaa [syötävät sienet]? = Ты умеешь распозновать [съедобные грибы]?
    3. Что-то нужно сделать с преметом:
      Tämä on [aamulla otettava lääke]. = Это [лекарство, которое нужно принять с утра].
      [Arkistoitavat dokumentit] ovat pinossa. = [Документы, которые нужно поместить в архив,] лежат в куче.
      Muista [nämä lukittavat ovet]! = Не забудь про [эти запираемые двери]! = Не забудь запереть эти двери!

    Пассивное причастие прошлого времени tU &

    → пассивная основа + -U

    Примеры
    1. oppia → opittu
      alkaa → alettu
    2. voida → voitu
    3. jutella → juteltu
    4. pelätä → pelätty
    5. tarvita → tarvittu
    6. pidetä → pidetty
    Примеры использования tU-причастия
    • valittu johtaja (выбранный лидер) = johtaja, joka valittiin (лидер, которого выбрали)
    • soitettu biisi (проигранная песня) = biisi, joka soitettiin (песня, которую проиграли)

    Агентное причастие mA &

    → сильная основа + -mA

    Примеры
    1. oppia → oppima
    2. juoda → juoma
    3. purra → purema
    4. kadota → katoama
    5. tarvita → tarvitsema
    6. vanheta → vanhenema
    Примеры использования mA-причастия
    • aloittamani kurssi (начатый мною курс) = kussi, jonka aloitin (курс, который я начал(а))
    • kirjoittamani kirja (написанная мною книга) = kirja, jonka kirjoitin (книга, которую я написал(а))

    Отрицательное причастие mAtOn &

    4. kadota → katoamaton
    5. tarvita → tarvitsematon
    6. vanheta → vanhenematon
